Project Overview

This academic project analyzes e-bike usage data collected from open NYC CitiBike systems. The primary goal is to understand urban micromobility patterns through hourly and daily breakdowns, user type segmentation, and trip length estimation.

Tools & Technologies

The dashboard uses HTML5, CSS3, and JavaScript along with Chart.js for data visualizations. JSON is used to load data dynamically. The project is deployed using GitHub Pages.

Academic Context

Developed as part of a dissertation project at Arden University for STU105154 (2025), this project highlights applied data visualization and user-centered dashboard design.
